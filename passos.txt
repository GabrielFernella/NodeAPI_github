8 - Baixando o container pelo Docker

8.3 - docker ps -a (mostra todas as imagens salvas no se container)
8.4 - docker start mongodb
9 - npm install mongoose (Dependencia para MongoDB)(ORM de bancos não relacionais) 
10 - npm install require-dir (opcional, faz o require em todos os arquivos automaticamente )
